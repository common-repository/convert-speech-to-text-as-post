== Description ==

#### Convert Speech To Text As Post
The complete option for speech to text converter in wordpress where you can speak you desire content to make it as post. You can upload an audio file to convert it into text and that text will be automatically insert in the post in which post you want to record and convert it as a post.With upload audio option it starts listening and converting it text as an audio file is uploaded.

**Convert Speech To Text As Post - Features:**

*   Create as many posts as you want - Convert Speech To Text As Post
*   Convert your speech to text as a post with one click - Convert Speech To Text As Post
*   Pause and start again from where you want - Convert Speech To Text As Post
*   Upload an audio file to convert it to text and add that text to post - Convert Speech To Text As Post
*   Upload tutorial like audio files to make it as a post - Convert Speech To Text As Post
*   Upload media files like mp3,mp4,flv all the audio features files to convert - Convert Speech To Text As Post

== Installation ==

Install Convert Speech To Text As Post either via the WordPress.org plugin directory, or by uploading the files to your server via FTP.

Activate the plugin through the 'Plugins' menu in WordPress.

Click on the posts from the wordpress menu and select the post in which you want to add text while speaking or by uploading the audio file to convert it text and insert that text to the post.
